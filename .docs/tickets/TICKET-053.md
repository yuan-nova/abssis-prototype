## Description

Generate reports on teaching activities including materials covered and teaching hours.

## Acceptance Criteria

- [ ] Report of all journals by teacher and period
- [ ] Show teaching materials and topics covered
- [ ] Calculate total teaching hours per teacher
- [ ] Calculate total teaching hours per subject
- [ ] Journal completion percentage
- [ ] Filter by teacher, subject, class, date range
- [ ] Export to Excel and PDF
- [ ] Include incomplete journal indicators
- [ ] Curriculum coverage analysis (optional)
- [ ] Comparison across teachers/classes

## Technical Notes

- Join journals with schedules for completeness
- Count actual teaching periods
- Group by various dimensions
